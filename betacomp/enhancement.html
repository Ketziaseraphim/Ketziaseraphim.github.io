Youtube Player

Description

Videoplayer can be used to enhance your website to become more impressive. As long the visitor's browser supports the video player, the video can be played properly.

You can add a video to your website using Youtube. By using iframe tag in your HTML codes and put the video source into it.

Audio Player

Description

This is an audio player in the website so people can play the audio player and hear the audio. The audio itself can be in .mp3, .ogg, and .wav formats. It can also be in various type of audio such as website backsound, music, recordings, podcasts, and many things.

You just need to put simple html code with audio tag and reference the source. The source can be both through internal storage or external which is online source.

Google Maps

Description

This is a feature to display certain area using google maps. This feature is great for a website that wants the visitors konw where to find the certain place. At the end people also can save its location and navigate through the website.

This Feature is using iframe tag and put a google maps source in it. It is free and no need to put a javascript into the code. It is very simple to implement the maps in the website page.
